# multipool_setup

Installation files for multipool-installer

### 系统要求
- Ubuntu 22.04 LTS
- 最小 2GB RAM（推荐 4GB+）
- 20GB+ 磁盘空间

### 安装说明
请访问 https://github.com/cryptopool-builders/Multi-Pool-Installer 获取完整安装指南

### 支持的功能
- YiiMP 单服务器/多服务器部署
- NOMP 服务器部署
- 守护进程钱包构建器
- 系统监控和管理
- 自动更新和维护

### 注意事项
- 仅支持全新安装的 Ubuntu 22.04 系统
- 请不要在生产环境中使用 root 用户运行
- 建议在安装前进行系统备份
